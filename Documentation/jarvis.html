Jarvis March Convex Hull Algorithm Documentation

Point class

This class represents a point in 2D space with x and y coordinates.

Left_index(points)

Finds the leftmost point in a list of points.

orientation(p, q, r)

Determines the orientation of an ordered triplet (p, q, r). It returns:

  • 0 if p, q, and r are collinear
  • 1 if they are clockwise
  • 2 if they are counterclockwise

convexHull(points, n)

Computes the convex hull of a set of points using the Jarvis March algorithm. It starts from the leftmost point and iteratively finds the next point on the hull. This function returns the convex hull as a list of coordinates.

Jarvis(a)

Main function implementing the Jarvis March algorithm to find the convex hull. It takes a list of points, transforms them into Point objects, and then computes the convex hull using the convexHull function. The final convex hull is returned as a list of coordinates.
